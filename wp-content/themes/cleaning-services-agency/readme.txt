== Description ==

Cleaning Services Agency is a purpose-built and visually appealing website template designed to cater to cleaning companies, janitorial services, and home maintenance businesses. Tailored to meet the unique needs of the cleaning industry, this theme offers a comprehensive set of features that facilitate effective online representation and customer engagement for cleaning service providers. Ideal for a range of users, from small local cleaning businesses to larger agencies, the Cleaning Services Agency theme offers a platform to showcase cleaning services, highlight expertise, and attract potential clients. Its user-friendly interface and seamless integration with WordPress make it accessible to users with varying technical expertise, enabling them to establish a professional online presence with ease. The standout benefit of the Cleaning Services Agency theme is its industry-specific design and functionality. As a purpose-built theme, it offers elements like service showcase sections, client testimonials, before-and-after images, and interactive contact forms tailored to cleaning service providers. This ensures that visitors can quickly understand the services offered and easily get in touch for inquiries or bookings. Additionally, the theme incorporates features such as pricing tables, service packages, and booking integration, streamlining the process for potential clients to select services and schedule appointments directly from the website. This enhances customer convenience and contributes to a seamless user experience.

== Homepage Installation ==

Step-1. Go to Dashboard -- Pages -- Add new page -- Add templates (Home Page) -- Publish it.
Step-2. Go to Dashboard -- Settings -- Reading -- Click on static page -- Select home page -- Save changes.

Header Section
Step-1. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Header Section -- Fill the contact information. -- Publish it.

Slider Section
Step-1. Go to Dashboard -- Posts -- Add new Category -- Add new post -- Add title, content, featured image and select category -- Publish it.
Step-2. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Slider Section -- Select the category -- Publish it.

About Section
Step-1. Go to Dashboard -- Pages -- Add new about page -- Add title, content-- Publish it.
Step-2. Go to Dashboard -- Posts -- Add new Category -- Add new post -- Add title, content, featured image and select category -- Publish it.
Step-3. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage About Section -- Select about page and service category -- Publish it.

Footer Section
Step-1. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Footer Section -- Add copyright text -- Publish it.
